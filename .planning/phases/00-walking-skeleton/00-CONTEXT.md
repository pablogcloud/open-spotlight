# Phase 0 Context

## Goal

Prove that a native macOS launcher can offer one coherent, safe, polished streaming experience over Claude Code, Codex CLI, and Grok Build CLI using the user's existing CLI authentication.

## Design read

Surface: global command/search overlay. Audience: technical, design-conscious Mac users. Vibe: restrained, precise, tactile, fast. Anchor: Raycast interaction quality, interpreted through native macOS materials rather than cloned. Constraints: keyboard-first, privacy disclosure, reduced motion, provider failure visibility.

## Scope boundary

No persistent semantic index, marketplace, cloud sync, autonomous file edits, or provider auto-routing in this phase.
