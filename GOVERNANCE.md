# Governance

Open Spotlight is currently founder-led while the project is pre-alpha.
Maintainers review roadmap changes, provider compatibility, privacy boundaries,
security fixes, releases, and use of project or provider marks.

Normal development happens through reviewed pull requests. Changes affecting
credentials, disclosure, signing, updates, telemetry, governance, licensing, or
public claims require explicit maintainer approval and recorded verification.

The final repository owner, maintainer list, decision process, contribution-rights
model, and trademark owner remain publication gates.
