# Governance

Open Spotlight is currently founder-led by `pablogcloud` while the project is
pre-alpha. Maintainers review roadmap changes, provider compatibility, privacy
boundaries, security fixes, releases, and use of project or provider marks.

Normal development happens through reviewed pull requests. Changes affecting
credentials, disclosure, signing, updates, telemetry, governance, licensing, or
public claims require explicit maintainer approval and recorded verification.

The repository is personally owned and maintained by `pablogcloud`. Contributions
use the Apache-2.0 contribution terms described in `CONTRIBUTING.md`. Governance
may be expanded as the maintainer community grows.
